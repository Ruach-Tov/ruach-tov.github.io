№ 9: The OAuth 400 That Said Only "Error"

On March 16, 2026, Anthropic OAuth users started getting 400 errors with a response body of just {"message": "Error"}. We diagnosed it through proxy forensics, documented it for Context Hub, and built an MCP server to serve the fix.
